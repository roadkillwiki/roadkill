Roadkill .NET Wiki engine
www.roadkillwiki.org
Version 1.1

Pre-requisites:
------------------------
* .NET Framework 4.0
* MVC 3 Razor (via the ASP.NET Web Installer: http://go.microsoft.com/fwlink/?LinkId=210269)
* IIS7+ is preferred, the app has not been tested with IIS6


Installation
------------------------

1) Create your website or virtual directory in IIS
2) Copy the contents of this folder to the directory
3) Point your browser to the URL - the installer will do the rest for you.

Please report any bugs, issues or suggestions at http://www.bitbucket.org/mrshrinkray/roadkill/issues
